The Eclipse SCADA Admin Client is split up in several perspectives. Each perspective focuses on one particular topic of the Eclipse SCADA system.

The following perspectives are available:

Client Perspective
The client perspective shows only minimal functionality. It acts as an entry point to all other functionality.
DA Perspective
The DA (DataAccess) perspective covers the Eclipse SCADA DA Interface. This is what was previously the the Eclipse SCADA Test Client.
Chart Perspective
The chart perspective replaces the HD (Histroical Data) perspective and provides chart views for HD and DA at the same time.
AE Perspective
The AE (Alarms&Events) perspective allows peeking into the alarms and events system.
